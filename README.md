# Bruma

Bruma es una tienda online desarrollada con React y Vite, inspirada en una cafetería de especialidad. La aplicación permite explorar productos, agregarlos al carrito y simular una experiencia de compra moderna y minimalista.

# Características

- Diseño moderno y responsive.
- Catálogo de productos.
- Carrito de compras dinámico.
- Navegación entre páginas con React Router.
- Secciones destacadas y categorías.
- Página de contacto.
- Página "Sobre nosotros".
- Checkout simulado.
- Componentes reutilizables.

# Tecnologías utilizadas

- React
- Vite
- JavaScript
- CSS Modules
- HTML

# Instalación

Cloná el repositorio:

```bash
git clone https://github.com/tu-usuario/bruma.git
```

Entrá en la carpeta del proyecto:

```bash
cd bruma
```

Instalá las dependencias:

```bash
npm install
```

Iniciá el servidor de desarrollo:

```bash
npm run dev
```

Para generar la versión de producción:

```bash
npm run build
```

# Bruma Coffee

Proyecto de tienda online de café desarrollado como práctica de Front-End, enfocado en la experiencia de usuario, el diseño visual y la simulación de un flujo de compra.

La aplicación presenta distintos blends de café, información institucional y un carrito de compras con interacción dinámica.

# Descripción

Bruma Coffee es una página web inspirada en una cafetería boutique, donde el usuario puede explorar diferentes variedades de café, conocer sus características y agregarlas a un carrito de compras.

El proyecto busca reflejar una estética cálida y moderna, alineada con la identidad del café, y aplicar conceptos fundamentales de desarrollo web.

# Funcionalidades

- Página de inicio con presentación visual del producto
- Listado de productos con nombre y precio
- Botón Agregar al carrito
- Vista de detalle del producto
- Carrito de compras interactivo
- Modal de inicio de sesión
- Sección "Acerca de nosotros" (misión y visión)
- Sección de contacto
- Diseño responsive

# Tecnologías utilizadas

# Cómo ejecutar el proyecto

- Clonar el repositorio:

git https://github.com/tu-usuario/bruma-coffee.git

- Instalar dependencias:

npm install

- Ejecutar el servidor de desarrollo:

npm run dev

Abrir el proyecto en el navegador:

http://localhost:5173
